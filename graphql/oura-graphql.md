# Oura Ring GraphQL API

Oura Ring is a smart ring for sleep, recovery, and activity tracking. The API covers sleep stages, readiness scores, activity metrics, heart rate, HRV, SpO2, and workout detection for health and wellness integration.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Oura Ring REST API
**Documentation:** https://cloud.ouraring.com/docs/
