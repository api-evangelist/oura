# Oura Ring (oura)

Oura Ring is a smart ring health tracking platform that provides a REST API for accessing sleep, activity, readiness, heart rate, daily health scores, and 50+ biometric metrics via OAuth2. Developers can build integrations allowing users to share their Oura Ring data with third-party services, with support for webhooks for near real-time updates.
